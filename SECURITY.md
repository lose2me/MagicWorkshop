## 报告漏洞 (Reporting a Vulnerability)

如果您发现了本项目中的安全漏洞，请**不要**直接在 GitHub Issues 中公开提交，以免被恶意利用。

请通过以下方式联系维护者：

1.  **发送邮件**: 请将漏洞详情发送至 `lw4289@gmail.com`。
2.  **私信联系**: 您也可以通过 Bilibili 私信联系 泠萌404 `UID:136850`。

### 报告内容应包含：
*   漏洞的类型和严重程度。
*   复现漏洞的详细步骤或代码示例。
*   受影响的版本。

我们会在收到报告后的 **48 小时内** 确认收到，并尽快评估和修复。

## 依赖项安全 (Dependencies)

本项目依赖于多个第三方开源组件（如 FFmpeg, ab-av1, PyQt6 等）。
*   如果是这些上游组件的安全问题，建议直接向对应的上游项目报告。
*   我们会定期更新依赖项以修复已知的安全漏洞。
